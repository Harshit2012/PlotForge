About

PlotForge turns data enter by user into graph. Area graph, Pie graph, Line graph and Bar graph are currently available here. Create this website using Chart.js.

Features:-

One click to create

By clicking to one button you can create the graph, just you have to enter the data.



One click to clear

By clicking one button you can clear the data from the website.



One click to download

By clicking one button you can download the graph from the website.
